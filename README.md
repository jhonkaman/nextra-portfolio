# Nextra Portfolio

## Create

```
npm init -y
```

```
npm i next react react-dom nextra nextra-theme-docs
```

```
touch next.config.mjs mdx-components.js
```

```
mkdir app
```

```
touch app/layout.jsx
```

```
mkdir app/[[...mdxPath]]
```

```
touch app/[[...mdxPath]]/page.jsx
```

```
mkdir content
```

```
touch content/index.mdx
```

## Start dev server

```
npm run dev
```
